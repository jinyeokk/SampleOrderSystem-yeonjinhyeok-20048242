# Phase 4 : 모니터링

## 1. 화면 구성

모니터링은 단일 화면으로 구성되며 메뉴 선택 즉시 전체 현황을 출력한다.

```
╔══════════════════════════════════════╗
║            모니터링                   ║
╠══════════════════════════════════════╣
║  1. 현황 새로고침                    ║
║  0. 메인 메뉴로 돌아가기             ║
╚══════════════════════════════════════╝
```

---

## 2. 기능 상세

### 2.1 상태별 주문 수 요약

```
──────────────────────────────────────────────────────────────
  주문 현황 요약                         (2026-05-08 15:30:00)
──────────────────────────────────────────────────────────────
  RESERVED    PRODUCING    CONFIRMED    RELEASE
  ────────    ─────────    ─────────    ───────
     3건          2건          5건         12건
──────────────────────────────────────────────────────────────
```

- `REJECTED` 상태는 집계에서 제외
- 우측 상단에 현재 시각 표시

---

### 2.2 시료별 재고 현황

```
──────────────────────────────────────────────────────────────
  시료별 재고 현황
──────────────────────────────────────────────────────────────
 ID        시료명                재고    생산중    대기주문
──────────────────────────────────────────────────────────────
 A-001     갈륨비소 웨이퍼        120       0         0
 A-002     실리콘 기판              15      27         1
 B-001     인화인듐 박막             0      54  ⚠      2
──────────────────────────────────────────────────────────────
```

| 컬럼 | 설명 |
|------|------|
| 재고 | 현재 가용 재고 수량 |
| 생산중 | 현재 `PRODUCING` 주문의 생산 필요 수량 합계 |
| 대기주문 | 해당 시료의 `RESERVED` 상태 주문 건수 |
| ⚠ 표시 | 재고 0이고 생산중인 시료 |

---

## 3. 데이터 집계 로직

```
모니터링 집계:

1. 상태별 주문 수
   orders.filter(status != REJECTED)
   .groupBy(status)
   .count()

2. 시료별 재고 현황
   for each sample:
     재고        = sample.stock
     생산중 수량 = sum(production_queue.required_qty
                      WHERE order.sample_id == sample.id
                      AND   production_queue.status != DONE)
     대기 주문   = orders.count(status=RESERVED, sample_id=sample.id)

3. 최근 활동 이력
   order_history.orderBy(changed_at DESC).limit(10)
```
