## 2. 기능 상세

### 2.1 시료 등록

#### 입력 흐름

```
──────────────────────────────────────
  시료 등록
──────────────────────────────────────
시료 ID      >> A-001
시료명       >> 갈륨비소 웨이퍼
평균 생산시간 (min/ea) >> 45
수율 (0.01 ~ 1.00)   >> 0.92

──────────────────────────────────────
  입력 내용 확인
──────────────────────────────────────
  시료 ID        : A-001
  시료명         : 갈륨비소 웨이퍼
  평균 생산시간  : 45 min/ea
  수율           : 0.92 (92%)
──────────────────────────────────────
등록하시겠습니까? (Y/N) >> Y

[완료] 시료 'A-001 갈륨비소 웨이퍼' 이(가) 등록되었습니다.
```

#### 입력 필드 및 검증 규칙

| 필드 | 입력 형식 | 검증 규칙 |
|------|-----------|-----------|
| 시료 ID | 문자열 | 필수, 중복 불가, 최대 20자 |
| 시료명 | 문자열 | 필수, 최대 50자 |
| 평균 생산시간 | 숫자 (min/ea) | 필수, 0 초과 실수 |
| 수율 | 소수 | 필수, 0.01 이상 1.00 이하 |

#### 초기 재고

- 등록 시점 재고는 `0 ea`으로 시작
- 재고는 생산 완료 시 자동 증가

#### 오류 케이스

| 오류 | 메시지 |
|------|--------|
| 중복 ID | `[오류] 이미 존재하는 시료 ID입니다: A-001` |
| 수율 범위 초과 | `[오류] 수율은 0.01 ~ 1.00 사이 값이어야 합니다.` |
| 생산시간 ≤ 0 | `[오류] 평균 생산시간은 0보다 커야 합니다.` |

### 2.2 시료 조회

#### 화면 출력

```
────────────────────────────────────────────────────────────
  시료 목록 (총 3건)
────────────────────────────────────────────────────────────
 ID        이름                평균생산시간  수율      재고
────────────────────────────────────────────────────────────
 A-001     갈륨비소 웨이퍼      45 min/ea   92%    120 ea
 A-002     실리콘 기판           30 min/ea   88%     45 ea
 B-001     인화인듐 박막        120 min/ea   75%    0 ea ⚠
────────────────────────────────────────────────────────────
```

#### 컬럼 구성 (합계 60자)

| 컬럼 | 너비 | 정렬 |
|------|------|------|
| ID | 10 | 좌 |
| 이름 | 20 (display) | 좌 |
| 평균생산시간 | 14 | 우 |
| 수율 | 6 | 우 |
| 재고 | 10 | 우 |

#### 표시 규칙

| 항목 | 표시 방식 |
|------|-----------|
| 수율 | 소수 → 퍼센트 변환 (0.92 → `92%`) |
| 평균 생산시간 | `{값} min/ea` 고정 형식 |
| 재고 | `{값} ea` 형식, 우측 정렬 |
| 재고 0 | `0 ea ⚠` 강조 표시 |

#### 정렬 기준

- 기본 정렬 : 시료 ID 오름차순

### 2.3 시료 검색

#### 입력 흐름

```
──────────────────────────────────────
  시료 검색
──────────────────────────────────────
검색어 (이름 / ID) >> 갈륨

──────────────────────────────────────
  검색 결과 (1건)
────────────────────────────────────────────────────────────
 ID        이름                평균생산시간  수율      재고
────────────────────────────────────────────────────────────
 A-001     갈륨비소 웨이퍼      45 min/ea   92%    120 ea
────────────────────────────────────────────────────────────
```

#### 검색 규칙

| 항목 | 내용 |
|------|------|
| 검색 대상 | 시료 ID, 시료명 |
| 매칭 방식 | 부분 일치 (대소문자 무시) |
| 결과 없음 | `[안내] 검색 결과가 없습니다.` 출력 후 재검색 또는 복귀 |

## 4. 모델 정의

```python
@dataclass
class Sample:
    sample_id: str
    name: str
    avg_production_time: float  # min/ea
    yield_rate: float           # 0.01 ~ 1.00
    stock: int = 0

    def yield_percent(self) -> str:
        return f"{self.yield_rate * 100:.0f}%"

    def required_production(self, order_qty: int) -> int:
        return math.ceil(order_qty / self.yield_rate)
```
