# Phase 6 : 출고처리

## 1. 개요

메인 메뉴 **[6] 출고처리** 선택 시 서브메뉴 없이 출고 가능 주문 목록이 바로 표시된다.  
`CONFIRMED` 상태의 주문을 선택하면 즉시 출고 처리하여 상태를 `RELEASE`로 전환한다.

---

## 2. 화면 흐름

```
메인 메뉴 [6]
    │
    ▼
출고 가능 주문 목록 (CONFIRMED)
    │
    ├── 번호 선택 ──► 출고 처리 완료 화면 (CONFIRMED → RELEASE)
    │                      │
    │                 목록으로 복귀
    │
    └── 0 입력 ──► 메인 메뉴로
```

---

## 3. 화면 정의

### 3.1 출고 가능 주문 목록

`CONFIRMED` 상태 주문만 표시한다.

```
────────────────────────────────────────────────────────────
  출고 가능 주문 (총 3건) (CONFIRMED)
────────────────────────────────────────────────────────────
 NO  주문 번호              고객명              시료명        수량
────────────────────────────────────────────────────────────
  1  ORD-20260508-001    (주)반도체코리아    실리콘 기판     30 ea
  2  ORD-20260508-002    ABC전자             인화인듐 박막   20 ea
  3  ORD-20260508-003    XYZ소재             갈륨비소 웨이퍼 80 ea
────────────────────────────────────────────────────────────
출고할 번호 (0: 메인 메뉴) >> 
```

#### 표시 규칙

| 항목 | 내용 |
|------|------|
| 정렬 | 승인일시 오름차순 (오래된 순 우선) |
| 목록 없음 | `[안내] 출고 가능한 주문이 없습니다.` 후 메인 복귀 |

#### 컬럼 구성 (합계 60자)

| 컬럼 | 너비 | 정렬 |
|------|------|------|
| NO | 4 | 우 |
| 주문 번호 | 20 | 좌 |
| 고객명 | 16 (display) | 좌 |
| 시료명 | 14 (display) | 좌 |
| 수량 | 6 | 우 |

## 5. 상태 전이

```
CONFIRMED ──[출고 처리]──► RELEASE
              ↓
        released_at = datetime.now()
        order.status = RELEASE
```

---

## 6. 데이터 흐름

```
메인 메뉴 [6] 선택
    │
    ▼
release_menu.run(ctx)  ← 서브메뉴 없이 바로 목록 표시
    │
    ├─ release_service.get_pending_list()
    │   └─ order_repo.find_by_status(CONFIRMED)
    │      .sort(updated_at ASC)
    │
    └─ 번호 선택 → release_service.release_one(order_id)
                    order.status = RELEASE
                    order.released_at = now()
```
